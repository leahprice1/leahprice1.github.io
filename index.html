Welcome to my project!!
This is a website designed to showcase some HTML/Jquery effects and help exercise GUI fimiliarity On the next few tabs I'll go over those effects, tell you a bit about myself, and give you my contact information.


The main requirements for this project were to:
1. create a personal website with multiple tabs
2. Host the website on Github
3. Maintain good file and folder structure.
As you can see, I do in fact have a personal website with multiple tabs. It is in fact hosted on github, and there is good folder and file organization in my opinion. Obviously, AI helped a lot with this project because creating a website from scratch takes too long and I already know how to do that! I used AI as well to help me understand more about the Jquery effects and how to implement them into my website.

This tab displays all of the effects I used for my project! For more information open the accordian effect below to learn about each one.

Accordian

This is the currect effect you're on! This effects, and all others after, were taken from the jQuery user interface site shown in class. This creates sections that when opened, close the other sections above or below it (like an accordian). You can put whatever you want within these sections but for the purpose of this website and project, I'm only putting text.

Check Boxes

The check boxes are an effect that create multiple boxes with the ability to check and uncheck them, different labels can be created and it can be used to show certain selections and basically have any functionality that you can think of for checkboxes. For the purpose of this project and website it's just checking compliments that apply to me. :)

Slider

The slider effect is pretty self explanatory, it's just a box that slides left and right on a bar. This could also be used for lots of implimentations, basically anything you can think to use a slider for, for the purpose of this project it's just a normal slider.

Date Picker

This is also pretty self explanatory, this effect has a box and when you click it it allows you to pick a date by pulling up a little calendar. You can scroll through the months and years and specify which date you need. This doesn't have as wide of a range of functionality as the previous effects but its very useful.

Ellipse Changer

This effect was one I modified, it was originally used to change the color and size of a circle, but I decided it would be more interesting to change the width and height so that's what I did. Out of all the effects this one probably has the least functionality compared to the others but there are still some uses I could think of this being used for, like changing the dimensions of an image.
